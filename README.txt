Cameron's Bakery
A spec site for Cameron's Bakery, 4.22.2016

By Josh Overly

Description

Owner can add articles and information about their bakery business.  Users can see information and contact owner.

Setup/Installation Requirements

You will need to have MAMP or WAMP installed.

Go to https://github.com/jos-h20/cameron_bakery.
Copy the directory.
From your terminal, change your directory to desktop.
Type in git clone https://github.com/jos-h20/cameron_bakery
Open MAMP and click on Preferences
Click on the folder icon and select cameron_bakery
Start servers
Open WebStart Page
Select Tools/PHPMyAdmin
Click on Import
Choose file/go to cameron_bakery/sites/db-backup
Click on the zipped database
Click Go
Once database has downloaded, click bakery
Go to Privileges, click Add User
For user name put in cameron
change host to local
For the password, type in cameron
Click Go
In your browser, type in localhost:8888
Select standard installation/select English
Enter josh_bookstore_database, user name: cameron, password: cameron
Under advanced options, change host to 127.0.0.1
And port to 8889
The user name and password are the same for the site.
